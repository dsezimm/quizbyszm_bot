# 🤖 Programming Quiz Telegram Bot

A Telegram bot for testing your knowledge of Python 🐍 and SQL 🗄️.

## 🚀 Features
- 15 random questions per quiz
- Python & SQL topics
- Hints and explanations
- Score tracking
- Topic analysis

## ⚙️ Installation
1. Clone repo
2. Install requirements
3. Set BOT_TOKEN
4. Run bot
